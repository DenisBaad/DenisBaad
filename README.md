# 👩🏻‍💻 Denis Baad

**`Desenvolvedor FullStack`**

Desenvolvedor Full Stack graduado em Análise e Desenvolvimento de Sistemas e atualmente cursando pós-graduação em Engenharia de Software, com foco em entregar soluções completas, escaláveis e de alto valor. Sou motivado pelo aprendizado contínuo e por enfrentar desafios que gerem impacto positivo para pessoas e negócios.

No backend, atuo com C#, desenvolvendo APIs RESTful e microserviços orientados ao domínio (DDD), com foco em performance, segurança e escalabilidade. Possuo experiência com autenticação via JWT, testes automatizados, Docker para provisionamento de ambientes e Apache Kafka para mensageria e comunicação assíncrona entre serviços. Também trabalho com bancos de dados relacionais e não relacionais, realizando modelagem, otimização de consultas e definição de estratégias de persistência adequadas conforme o contexto da aplicação.

No frontend, utilizo Angular, HTML, CSS e TypeScript para desenvolver interfaces modernas, intuitivas e responsivas, aplicando boas práticas como lazy loading, route guards e prevenção de memory leaks.

Também possuo experiência com Git, pipelines de CI/CD e metodologias ágeis (Scrum), colaborando de forma eficiente em equipes multidisciplinares. Além disso, tenho familiaridade com serviços em nuvem, como Azure e AWS, incluindo integração entre serviços e consumo de dados em diferentes arquiteturas.
